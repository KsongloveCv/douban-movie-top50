## 示例 2：直接运行 douban_crawler.py

```bash
python douban_crawler.py
```

## 示例 3：自定义数量和输出路径

```bash
python douban_crawler.py --count 100 --output my_top100.json
```

## 示例 4：仅终端显示不保存文件

```bash
python douban_crawler.py --no-save
```

## 示例 5：增加重试次数（网络不稳定时）

```bash
python douban_crawler.py --retry 5
```
